# 📸 마침 (MaChim)

## 📌 서비스 개요
일상의 순간들을 **포토 덤프로 기록**하고 **공유**할 수 있는 웹 서비스입니다.  
사용자는 캘린더에 일정을 등록하고, 일정별로 사진을 업로드하여 자동으로 정리된 포토 덤프를 생성할 수 있습니다.

<br>

## 🔗 배포 URL
https://photo-dump-daylog-magic.lovable.app

<br>

## 👩🏻‍💻 페르소나

### 👤 페르소나 1: 민지 (25세, 직장인)
이름: 민지  
나이: 25세  
직업: 마케팅 매니저  
목표: 바쁜 일상 속 소소한 순간들을 예쁘게 기록하고 싶다.

> 민지는 일상의 소소한 순간들을 기록하는 것을 좋아한다.  
> SNS에 일상을 공유하는 것을 즐기지만, 매번 사진을 편집하고 정리하는 것이 번거롭다.  
> '마침'을 통해 자동으로 예쁘게 정리된 포토 덤프를 만들어 공유하고 싶어 한다.

#### 📱 시나리오
- 하루를 마무리하며 오늘의 일정과 사진들을 기록한다.
- 자동으로 정리된 포토 덤프를 SNS에 공유한다.
- 과거의 기록들을 날짜별로 찾아보며 추억을 회상한다.

#### ✅ 요구사항
- 일정과 사진을 쉽고 빠르게 등록할 수 있어야 한다.
- 자동으로 예쁘게 정리된 포토 덤프를 생성할 수 있어야 한다.
- 과거의 기록을 쉽게 찾아볼 수 있어야 한다.

#### 🎯 사용자 스토리
> 바쁜 직장인으로서 나는 일상의 순간들을 쉽고 예쁘게 기록하고 싶다.  
> 그래야 특별하지 않은 날들도 의미있게 남길 수 있기 때문이다.

#### 🚀 인수조건
- **Given** 사용자가 일정을 등록하고 사진을 업로드했을 때
- **When** 포토 덤프 생성 버튼을 클릭하면
- **Then** 자동으로 정리된 포토 덤프가 생성되어야 한다.

### 👤 페르소나 2: 준호 (32세, 프리랜서 여행 작가)
이름: 준호  
나이: 32세  
직업: 프리랜서 여행 작가  
목표: 여행의 순간들을 체계적으로 기록하고 공유하고 싶다.

> 준호는 여행 작가로 활동하며 다양한 장소를 다닌다.  
> 매일 수많은 사진을 찍지만, 이를 정리하고 관리하는 것이 어렵다.  
> '마침'을 통해 여행 일정별로 사진을 정리하고 포토 덤프로 만들어 기록하고 싶어 한다.

#### 📱 시나리오
- 여행 일정을 등록하고 그날의 사진들을 업로드한다.
- 여행지별로 포토 덤프를 생성하여 저장한다.
- 생성된 포토 덤프를 SNS나 블로그에 공유한다.

#### ✅ 요구사항
- 날짜별로 여행 일정과 사진을 구분해서 저장할 수 있어야 한다.
- 여러 장의 사진을 한 번에 업로드할 수 있어야 한다.
- 생성된 포토 덤프를 다양한 플랫폼에 공유할 수 있어야 한다.

#### 🎯 사용자 스토리
> 여행 작가로서 나는 매일의 여행 순간들을 체계적으로 정리하고 공유하고 싶다.  
> 그래야 나중에 콘텐츠로 활용하기 쉽기 때문이다.

#### 🚀 인수조건
- **Given** 사용자가 여행 일정을 등록했을 때
- **When** 해당 일정에 여러 장의 사진을 업로드하면
- **Then** 사진들이 일정별로 구분되어 저장되어야 한다.

<br>

### 👤 페르소나 3: 미영 (45세, 주부)
이름: 미영  
나이: 45세  
직업: 주부  
목표: 아이들의 성장 과정과 가족의 일상을 기록하고 싶다.

> 미영은 두 아이를 키우는 엄마다.  
> 아이들의 성장 과정과 가족의 일상을 기록하고 싶지만, 복잡한 앱은 사용하기 어렵다.  
> '마침'의 직관적인 인터페이스를 통해 쉽게 가족 앨범을 만들고 싶어 한다.

#### 📱 시나리오
- 아이들의 행사나 가족 일정을 캘린더에 기록한다.
- 찍은 사진들을 일정별로 정리한다.
- 만들어진 포토 덤프를 가족들과 공유한다.

#### ✅ 요구사항
- 직관적인 UI로 쉽게 일정과 사진을 등록할 수 있어야 한다.
- 자동으로 예쁘게 정리된 포토 앨범을 만들 수 있어야 한다.
- 가족들과 쉽게 공유할 수 있어야 한다.

#### 🎯 사용자 스토리
> 두 아이의 엄마로서 나는 아이들의 성장 과정과 가족의 일상을 쉽게 기록하고 싶다.  
> 그래야 소중한 추억들을 오래도록 간직할 수 있기 때문이다.

#### 🚀 인수조건
- **Given** 사용자가 가족 행사 일정을 등록했을 때
- **When** 해당 일정의 사진들을 선택하면
- **Then** 자동으로 예쁜 포토 앨범이 생성되어야 한다.

## 🚀 시작하기

### 설치

```bash
# 저장소 클론
git clone https://github.com/your-username/photo-dump-daylog-magic.git

# 디렉토리 이동
cd photo-dump-daylog-magic

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview
```
